# addr-paser

一个轻量的邮寄地址解析器

## 特性

1. 基于地址正则解析，简易轻便
2. 易于使用，经过生产环境数据测试，识别率在90%以上

## 用法

```php
$addr = "山西省运城市新绛县龙兴镇店头村店头大街1号 超哥特电话:13344556677 邮编：043100"; 
print_r((new AddrParser($addr))->parse());

Array
(
    [consignee] => 超哥特
    [mobile] => 13344556677
    [prov] => 山西省
    [city] => 运城市
    [dist] => 新绛县
    [addr] => 龙兴镇店头村店头大街1号
    [post_code] => 043100
)
```
